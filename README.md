# movie-interface-structure-ledger
该项目围绕在线电影与电视剧平台的界面结构展开整理，对首页模块排列逻辑、导航层级设计以及播放页信息区块构成进行系统记录。通过结构化文字归档建立资料库，用于研究不同影视网站在页面组织与结构设计方面的演变趋势。仓库仅保存公开页面资料，不包含影视资源内容或在线播放功能。
